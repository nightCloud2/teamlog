판매/나눔 사이트 시나리오 작성

1. 로그인 하지 않은 상태
판매/나눔 사이트로 이동 -> 전체로 게시글이 나오는지 확인
-> 페이지 이동 확인 -> 게시글 클릭 -> 목록으로 돌아가기 -> 글쓰기 클릭 -> 로그인페이지로 이동

2. 로그인 한 상태
2.1. 판매글 id와 로그인 id가 동일한 경우
가입시 자신이 기입한 동네로 옵션이 뜨는지 확인 ->
글쓰기 클릭 -> 글 작성 -> 유효성 체크되는지 보여줌-> 글 쓰기 완료 -> 자신의 글에 수정/삭제가 보이는지 확인 -> 글 수정 -> 판매상태 변경 -> 목록으로 돌아가서 상태 변경되었는지 확인 -> 글 삭제

2.2. 판매글 id와 로그인 id가 동일하지 않은 경우
판매글 클릭 -> 수정/삭제 보이지 않는 것 확인
