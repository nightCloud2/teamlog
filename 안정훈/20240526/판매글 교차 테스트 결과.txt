--- 판매글 교차 테스트 결과 ---

[[[ 판매글 목록 ]]]
1. 카테고리 선택하여 리스트 볼때 글 작성할 때 처럼
    "대분류 >중분류 > 소분류"로 보여주면 좋을 것같음				=> 보류
2. 주소 옵션값 value 변경하고 글 작성 하면 글작성 오류 발생			=> X

[[[ 판매글 작성 ]]]
1. radio버튼 글씨도 클릭했을 때 radio버튼이 눌릴 수 있도록 해줄 것		=> 완료
2. 이미지 등록에서 파일등록으로 되어있는데 이미지라고 명확하게 해줄 것	=> X
3. 거래방법 2개 이하로 제한 둔 부분 3개도 가능하면 좋을 것 같음		=> 컬럼 수 때문에 안됨
4. 상품명/내용 작성할 때 기준이 정해지면 좋을 것 같음(
    - 최소 길이(한글자만 작성 가능한게 이상한 것 같음				=> 제목(2글자), 내용(2글자)
    - 숫자/특수문자로만 들어간 경우							=> 고민
    - 공백인 경우											=> 완료(제목,내용)
5. 태그 글자수 프론트단에서 길이 조절							=> 완료

[[[ 판매글 수정 ]]]
1. 선택되어있던 카테고리가 됐다 안됐다가 나옴
   (대분류 > 중분류 > 소분류 select option)

--- 공통 ---
[[[ url ]]]
1. url을 직접 기재하여 판매글 페이지 이동하는 경우 막기				=> 완료
2. url을 길게 쓰면 에러페이지로 넘어가지 않고 NumberFormatException 발생
    - 숫자 10억 이상/문자
